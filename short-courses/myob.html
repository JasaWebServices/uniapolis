MYOB Accounting software

中英文讲解

Learn how to set up correctly your organisation’s business files for most efficient use possible, as well as learn the right way to set up all the relevant accounts, tax codes, card files and inventory items.

Objectives

This is hands on workshop to enable participants to get started and begin using MYOB accounting software through the followings

  • How to install MYOB accounting software
  • Brush up basic accounting terminology
  • Overview of MYOB accounting software
  • MYOB navigation tools
  • How to create new data files
  • Set up Account List and opening balance
  • Set up card files – customer, supplier, employee and personal cards
  • Set up Sales details and assign customers opening balance
  • Set up Purchases details and assign suppliers opening balance
  • Bank reconciliation
  • General reports that related to opening balances
  • Back up and restore data files
  • Recording Journal Entries
  • Processing Banking Transactions
  • Sales Management
  • Purchase Management
  • Multi-currency environment
  • GST reporting
  • Preference & Security settings
  • Others: Recurring items, Jobs & Categories, Budget etc.
